# Forming Paws — Client Demo

Live demo of the Forming Paws platform: nonprofit, health-first dog-breeding matchmaking.

**Live site:** https://morrisstephon51.github.io/forming-paws/

- `index.html` — landing page (mission, how it works, roadmap)
- `app.html` — the app demo: dog profiles, breed/sex/distance filters, health-verified gating, match flow, chat
- Demo runs on seeded data; production backend (Supabase) is the next build phase.
- Published via the `gh-pages` branch — push site changes there to update the live link.
